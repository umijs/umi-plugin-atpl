# umi-plugin-atpl

Umi plugin for atpl, based on [atpl-loader](https://github.com/sorrycc/atpl-loader).

## Usage

Save dependencies.

```bash
$ npm install umi-plugin-atpl --save-dev
```

Add plugin in `.umirc.js`.

```
export default {
  plugins: [
    'umi-plugin-atpl',
  ],
}
```
